# Ejercicio 4: Catálogo de productos

## Objetivo
Utilizar `Dictionary<TKey, TValue>` para almacenar y manipular pares clave–valor.

## Enunciado
Crea un catálogo de productos utilizando un diccionario, donde:

- El código del producto represente la clave.
- El nombre del producto represente el valor.

El sistema debe permitir las siguientes operaciones:

- Agregar nuevos productos al catálogo.
- Buscar un producto a partir de su código.
- Eliminar productos del catálogo utilizando su código.

## Restricción
No utilizar LINQ.
