# Ejercicio 3: Ordenamiento según cantidad de letras

## Objetivo
Trabajar con `List<string>` y aplicar operaciones básicas sobre colecciones.

## Enunciado
Dada una lista genérica que contiene varios nombres, ordenar los elementos de la lista de forma ascendente según la cantidad de caracteres que posee cada nombre.

El nombre con menos letras debe aparecer primero y el nombre con más letras al final.

## Restricción
No se permite el uso de LINQ.
