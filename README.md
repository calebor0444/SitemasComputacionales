# Ejercicios vistos en el curso de programación de sistemas computacionales
Contiene los distintos ejemplos vistos en clase, así como la resolución de los laboratorios ordenados por clase.

##  ¿Cómo descargar los ejercicios?

### Opción recomendada:
1. Descargar GithuboDesktop: https://desktop.github.com/download/
2. Una vez instalado realizar el mapeo del repositorio copiando la url: https://github.com/calebor0444/SistemasComputacionales.git y apuntandola a una ruta en su maquina
3. Desde la herramienta se puede obtener la última versión cuando gusten.
   
### Opción 1: Descargar todo 
1. Haz clic en el botón verde **Code**
2. Selecciona **Download ZIP**
3. Descomprime el archivo en tu computadora

### Opción 2: Clonar con Git (recomendado)
```bash
git clone https://github.com/calebor0444/SistemasComputacionales.git
```

### Opción 3: Descargar ejercicios individuales
Navega a la carpeta del ejercicio y descarga los archivos específicos.
